    MSWL Project Management

    Introduction

    Notes and Exercises for Project Management Subject coursed in Master on Libre Software (Master Universitario en Software libre) at Universidad Rey juan Carlos.

    Requirements

    • LaTeX: to compile source code documentation files.
    • pdf2htmlEX: Convert pdf to html files by @coolwanglu.

    pdf2htmlEX Installation

    $ sudo add-apt-repository ppa:coolwanglu/pdf2htmlex
$ sudo apt-get update
$ sudo apt-get install pdf2htmlEX
$ pdf2htmlEX 001-final-report-floss-release.pdf

    Exercises

    Exercises writed using LaTeX and pdf out in published in master/exercises:

    • Technical infrastructure in KDE: LaTeX & pdf.
    • How to join a new project: LaTeX & pdf.
    • Umbrella projects and communities: LaTeX & pdf.
    • Discussions in open collaborative projects: LaTeX & pdf.
    • Bug reporting guidelines: LaTeX & pdf.
    • Hiring a community manager: LaTeX & pdf.
